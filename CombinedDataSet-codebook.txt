SubjectID
  - subject ID numbers are from UCI HAR Dataset file subject_test.txt and subject_train.txt data files
ActivityID
  - activity numerical ID are from UCI HAR Dataset file y_train.txt and y_test.txt
ActivityLabel
  - activity descriptive label created by matching the ActivityID to the UCI HAR Dataset file activity_labels.txt
